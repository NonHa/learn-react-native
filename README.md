使用 react native api [PanResponder](https://www.reactnative.cn/docs/panresponder)
PanResponder 类可以将多点触摸操作协调成一个手势。它使得一个单点触摸可以接受更多的触摸操作，也可以用于识别简单的多点触摸手势。
